HealthQR PWA Testing Steps

1) Install requirements
   pip install -r requirements.txt

2) Run app
   python app.py

3) Open desktop browser
   http://127.0.0.1:5000/login

4) Check PWA
   Chrome -> Right click -> Inspect -> Application
   - Manifest should show HealthQR Referral
   - Service workers should show activated

5) Desktop install
   Chrome/Edge address bar -> Install icon
   Or menu (three dots) -> Save and share -> Install page as app

6) Android phone testing
   - Connect phone and laptop to same Wi-Fi
   - Run app normally because app.py already uses host=0.0.0.0:
     python app.py
   - Find laptop IPv4 using:
     ipconfig
   - Open on phone:
     http://YOUR-LAPTOP-IP:5000/login
   - Chrome menu -> Add to Home screen

Important:
- 127.0.0.1 works only on your laptop, not on mobile.
- For real public install, deploy with HTTPS.
- Database and tables are unchanged. The same database/healthqr.db is used.
