MOBILE DOCTOR APP TESTING

1) Run Flask:
   python app.py

2) Open mobile doctor app login:
   http://127.0.0.1:5000/m/login

3) Mobile phone testing on same Wi-Fi:
   python app.py
   Then open on phone:
   http://YOUR-LAPTOP-IP:5000/m/login

4) Login:
   - Enter doctor registered mobile number
   - Default login is Secret Code
   - Use Switch to Password if required

5) After login:
   - Mobile doctor dashboard opens
   - Referral cards display in mobile app style
   - Filter From Date / To Date
   - Export PDF opens printable PDF page

IMPORTANT:
- Browser/WebView cannot automatically read SIM number.
- Android native APK can use phone-number hint API later.
- Admin and Hospital login remain in existing web login page.
